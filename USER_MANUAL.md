# 语音笔记 — 用户手册

## 概述

语音笔记是一款录音与转写 App，提供 **Android** 和 **iOS** 双平台原生版本。录制或导入音频后，自动将语音转为文字。

两个平台核心功能对等。Android 版额外支持在线/离线 LLM AI 总结功能。

| | Android | iOS |
|---|---|---|
| 最低系统版本 | Android 8.0 (API 26) | iOS 14.0 |
| 离线 ASR | ✅ | ✅（需 iOS 15.1+） |
| 在线 ASR | ✅ | — |
| AI 总结 | ✅ 在线/离线 | — |
| 后台录音 | ✅ 前台服务 + WakeLock | ✅ 后台音频模式 |

---

## 一、主界面

打开 App 后进入**首页**，顶部标题栏显示"语音笔记"。

| 区域 | 说明 |
|---|---|
| 今日记录 / 总记录 | 当天统计概览 |
| 最近记录 | 按时间倒序展示录音卡片，已转写的卡片有「已转写」标签 |
| 📜 右上角 | 进入历史记录 |
| ⚙ 右上角 | 进入设置 |
| ＋ 右上角 | 新建录音 |

点击任意录音卡片进入**详情页**。

---

## 二、新建录音

### iOS

点击首页右上角 **＋** 按钮，**直接进入录音状态**，无需填写任何表单：

- 录音标题自动生成（格式："新录音 6月25日 14:30"）
- 界面顶部变红，显示脉冲红点 + 计时器
- 下方区域实时滚动显示语音转写文本
- 锁屏或切换到其他 App 时录音不会中断

也可点击右上角导入按钮，从手机存储中选择音频文件导入。

### Android

点击新建录音按钮后，可填写可选信息：

- **标题**：录音名称，留空则自动生成（格式："新录音 6月25日 14:30"）
- **备注**：补充说明，会显示在历史卡片副标题
- **说话人**：逗号分隔的参与人员姓名

点击 **开始录音** 按钮进入录音状态，或点击 **导入音频** 从本地选取音频文件。

### 结束录音（双平台通用）

点击底部红色 **结束** 按钮，App 自动：

1. 停止录音并保存音频文件
2. 完成语音转写
3. 返回首页
4. （Android）自动调用 AI 生成结构化总结

### 导入音频

如果已有录音文件，可点击导入按钮，从手机存储中选择音频文件。导入后会后台启动转写处理。

## 三、录音详情

从首页或历史记录点击录音卡片进入详情页。

### iOS 详情页

页面顶部有两个分段：

#### 3.1 音频

- 显示录音基本信息（标题、备注、时间、时长）
- **音频回放**：播放/暂停、快退 15 秒、快进 15 秒、拖动进度条
- 可分享/导出音频文件

#### 3.2 转写

显示语音转写文本的状态和内容：

| 状态 | 说明 |
|---|---|
| 转写准备中... | 等待转写开始 |
| 正在转写... | 转写进行中 |
| 转写失败 | 显示错误信息，可点击重试 |
| 转写完成 | 可点击查看全文，可导出分享 |

转写失败或不满意时可点击 **重新转写**。

### Android 详情页

页面顶部有三个分段：音频 / 转写 / 总结。

总结 Tab 显示 AI 结构化总结：

- **议题**：本次录音讨论的主要话题
- **结论**：达成的共识或决定
- **待办**：任务内容、负责人、截止时间
- **后续**：下一步跟进计划

总结失败时可 **重新总结**，完成后可 **导出** 分享。

## 四、设置

首页右上角 ⚙ 进入设置页。

### iOS 设置

#### 4.1 语音识别模型

选择 SenseVoice 模型质量：

| 模型 | 大小 | 适用场景 |
|---|---|---|
| INT8 | ~170 MB | 推荐，速度快，精度良好 |
| FP32 | ~860 MB | 精度更高，需要足够内存 |

> ⚠️ 设备内存低于 4GB 不建议使用 FP32。

**获取模型**：

| 方式 | 说明 |
|---|---|
| App 内下载 | 点击「下载」按钮，自动下载并解压 |
| 电脑下载后导入 | 用电脑下载 `.tar.bz2` 文件，传到手机上点「上传」导入 |

模型直链：

| 精度 | 下载地址 |
|---|---|
| INT8 (~170MB) | `https://github.com/k2-fsa/sherpa-onnx/releases/download/asr-models/sherpa-onnx-sense-voice-zh-en-ja-ko-yue-int8-2025-09-09.tar.bz2` |
| FP32 (~860MB) | `https://github.com/k2-fsa/sherpa-onnx/releases/download/asr-models/sherpa-onnx-sense-voice-zh-en-ja-ko-yue-2025-09-09.tar.bz2` |

下载完成后界面显示「模型已就绪」。

**模型生命周期**：App 启动时自动加载语音模型，直至 App 关闭才释放。录音之间不会重复加载/卸载。

> 首次启动如未安装模型，会自动弹出引导对话框，点击前往设置即可下载。

#### 4.2 标点符号模型

可选下载，用于给转写文本自动添加标点符号。

| 文件名 | 下载地址 |
|---|---|
| `sherpa-onnx-punct-ct-transformer-zh-en-vocab272727-2024-04-12.tar.bz2` | `https://github.com/k2-fsa/sherpa-onnx/releases/download/punctuation-models/sherpa-onnx-punct-ct-transformer-zh-en-vocab272727-2024-04-12.tar.bz2` |

> 模型为单个 tar.bz2 归档文件（约 1MB），App 内点击「下载」可自动下载并解压提取 `punct_ct_transformer.onnx`。也支持本地导入 `.tar.bz2` 归档或 `.onnx` 文件。

### Android 设置

#### 4.3 语音识别（ASR）

支持在线/离线模式独立切换。

**在线模式**：填写 FunASR WebSocket 地址，例如 `ws://192.168.1.100:10095`。需自行部署 FunASR 服务端。

**离线模式**：选择 INT8 / FP32 模型，下载方式同 iOS。

#### 4.4 AI 总结（LLM）

支持在线/离线双模式。

**在线模式**：填写 OpenAI 兼容 API 信息：

| 配置项 | 说明 | 示例 |
|---|---|---|
| API 地址 | OpenAI 兼容的 base URL | `https://api.deepseek.com` |
| API Key | 服务商提供的密钥 | `sk-xxxxxxxx` |
| 模型 | 模型名称 | `deepseek-v4-pro`、`gpt-4o-mini` |

**离线模式**：选择 Qwen2.5 GGUF 模型：

| 模型 | 大小 | 适用场景 |
|---|---|---|
| Qwen2.5-0.5B | ~352 MB | 低内存设备（< 3GB），速度快 |
| Qwen2.5-1.5B | ~986 MB | 内存充足设备，质量更好 |

模型直链：

| 模型 | 下载地址 |
|---|---|
| Qwen2.5-0.5B | `https://modelscope.cn/models/qwen/Qwen2.5-0.5B-Instruct-gguf/resolve/master/qwen2.5-0.5b-instruct-q4_k_m.gguf` |
| Qwen2.5-1.5B | `https://modelscope.cn/models/Qwen/Qwen2.5-1.5B-Instruct-GGUF/resolve/master/qwen2.5-1.5b-instruct-q4_k_m.gguf` |

#### 4.5 保存

配置完成后务必点击 **保存** 按钮。

## 五、历史记录

首页右上角 📜 进入历史记录页。

- **搜索**：输入关键词，按标题、备注、内容模糊匹配
- **删除单条**：Android 侧滑删除 / iOS 左滑删除
- **清空全部**：右上角删除图标，确认后删除所有记录及对应音频文件

## 六、常见场景速查

### iOS

| 场景 | 需准备 |
|---|---|
| 离线录音 + 转写 | 下载 SenseVoice ASR 模型 |
| 完整离线使用 | ASR 模型 + 标点模型（可选） |

### Android

| 场景 | ASR 模式 | LLM 模式 | 需准备 |
|---|---|---|---|
| 完全离线使用 | 离线 | 离线 | ASR 模型 + LLM 模型 |
| 公司内网环境 | 在线 | 在线 | FunASR 服务 + LLM API |
| 离线录音 + 在线总结 | 离线 | 在线 | ASR 模型 + LLM API Key |
| 在线转写 + 离线总结 | 在线 | 离线 | FunASR 服务 + LLM 模型 |

## 七、平台差异说明

| | Android | iOS |
|---|---|---|
| 新建录音 | 填写表单 → 开始录音 | 点击 ＋ 直接开始录音 |
| 录音后台保活 | 前台服务通知 + WakeLock | 后台音频模式 (Background Modes) |
| ASR 模式 | 在线 / 离线 | 离线（SenseVoice） |
| AI 总结 | 在线 / 离线 LLM | — |
| 详情页 Tab | 音频 / 转写 / 总结 | 音频 / 转写 |
| 设置项 | ASR + LLM + 连接测试 | ASR 模型 + 标点模型 |
| 权限提示 | 安装时声明，运行时弹窗 | 首次使用弹窗请求麦克风 |
| 模型生命周期 | 按需加载/释放 | 启动加载，关闭释放 |
| VAD 模型 | 按需下载 | 内置打包 |
| 离线 ASR 框架 | Sherpa-ONNX JNI | Sherpa-ONNX XCFramework |
| 数据存储 | Room (SQLite) + DataStore | Core Data + UserDefaults |

## 八、注意事项

- 录音过程中请通过「结束」按钮正常停止，勿强制关闭 App
- 离线 ASR 模型约 170~860MB，下载请使用 Wi-Fi
- 长时间录音（> 1 小时）会触发低电量提醒
- 删除录音记录会同时删除音频文件和转写文件，请确认后操作
- iOS 首次使用需允许麦克风权限
- iOS 首次启动如未安装 ASR 模型，会弹出引导对话框
